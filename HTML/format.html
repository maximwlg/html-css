Теги форматирования

Форматирование в HTML бывает логическим и физическим. Физическое форматирование просто изменяет внешний вид текста, логическое же позволяет разметить текст по смыслу и лишь потом - изменить его внешний вид. То есть при логическом форматировании внешний вид текста меняется не сам по себе, а потому что он наделен каким-то дополнительным смыслом.

Как создать сайт?

Для того, чтобы самостоятельно создать сайт, для начала в Блокноте нужно написать следующие теги:

<html> - начало документа
<head> - начало служебного раздела
</head> - закрытие служебного раздела
<body> - начало раздела с содержимым
</body> - закрытие раздела с содержимым
</html> - конец документа
